# ToDo List Web App

A simple todo list web app having the following features:-<br/>
  -Add new task<br/>
  -Delete task<br/>
  -Mark as complete<br/>
  -Displays list of all tasks<br/>
  -List of completed tasks<br/>
  -List of pending tasks<br/>
  <br/>
Created using:-<br/>
-HTML<br/>
-CSS<br/>
-JavaScript<br/>
